A simple way to display AdSense banners in your widgets

== Description ==

This plug-in is designed to help you to display AdSense banners in your widgets.

== Installation ==

1. Upload the plugin files to the <code>/wp-content/plugins/simple-adsense-widget</code> directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the Settings -> Simple AdSense widget screen to configure the plugin.
1. Enter your AdSense ID.
1. Optionally, enter your Google Analytics ID.

== Frequently Asked Questions ==

= How to choose the size of the banner? =

In this first version, banners are set in a "One size fits all" Automatic mode. This mode is particularily suited for responsive design templates.
